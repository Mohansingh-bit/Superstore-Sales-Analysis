# 📊 Superstore Sales Data Analysis

An exploratory data analysis (EDA) project using a real-world retail dataset with 9,000+ records. Built with Python using Pandas, Matplotlib, Seaborn, and Plotly.

---

## 🚀 Features

- Data cleaning — handles missing values and duplicates
- Key business insights — best category, region, and sub-category
- 4 visualizations — bar chart, box plot, interactive scatter, horizontal bar
- Summary report exported to CSV

---

## 🛠️ Technologies Used

- **Python**
- **Pandas** — data loading, cleaning, aggregation
- **Matplotlib** — bar charts
- **Seaborn** — box plot for distribution analysis
- **Plotly** — interactive scatter chart

## ⚙️ How to Run

### 1. Clone the repository
```
git clone https://github.com/Mohansingh-bit/Superstore-Sales-Analysis
cd superstore-analysis
```

### 2. Install dependencies
```
pip install -r requirements.txt
```

### 3. Run the analysis
```
python main.py
```
